# Screenshot Guide

Add screenshots in this folder using the exact file names below so they render automatically in the root README.

Required filenames:

- frontend.png
- argocd-dashboard.png
- kubernetes-resources.png
- azure-aks-acr.png

Recommended captures:

1. frontend.png
   - Main dashboard or landing page
   - Browser full-width capture

2. argocd-dashboard.png
   - Argo CD Application view showing Sync and Health status

3. kubernetes-resources.png
   - `kubectl get pods,svc` output or Kubernetes dashboard resources list

4. azure-aks-acr.png
   - Azure Portal view of AKS cluster and ACR repository/tags

Image tips:

- Use PNG format for clarity
- Keep width around 1600px
- Avoid sensitive information (tokens, secrets, subscription IDs)
